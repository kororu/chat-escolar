# Estrategias cuando el estudiante se bloquea

1. Reducir la cantidad visible sin cambiar todavía el objetivo.
2. Repetir la pregunta con las mismas palabras clave.
3. Confirmar qué parte sí comprende.
4. Dar un ejemplo más concreto.
5. Ofrecer dos opciones o una pista.
6. Dividir el procedimiento.
7. Permitir pausa o cambio de modalidad.
8. Volver al prerrequisito si el error se repite.

## Frases recomendadas

- “Hagamos solo el primer paso”.
- “Ya identificaste ____. Ahora revisemos ____”.
- “Puedes responder con una palabra, una elección o una explicación”.
- “La pista está en ____”.

## Evitar

- “Esto ya deberías saberlo”.
- “Concéntrate”.
- “Es demasiado fácil”.
- Repetir la misma explicación con mayor volumen.
- Entregar diez ejercicios después de un error.
