# Tema: Estrategias para comprender antes, durante y después de leer

## Respuesta breve

Comprender es construir significado mientras se lee. Para lograrlo se activa lo que ya se sabe, se formulan preguntas, se aclaran palabras, se relee y se resume.

## Qué aprenderás

- Comprender y aplicar estrategias para comprender antes, durante y después de leer en situaciones apropiadas para 5° básico.

## Objetivos de Aprendizaje relacionados

- LE05 OA 02

> Los códigos se usan como referencia de alineación. La formulación completa y vigente debe comprobarse en el portal oficial antes de una publicación normativa.

## Ideas fundamentales

- Antes se anticipa el tema y el propósito.
- Durante se comprueba, pregunta y relee.
- Después se sintetiza y responde con evidencia.
- Una estrategia se elige según la dificultad.

## Conceptos importantes

### Propósito

razón por la que se lee.
### Predicción

anticipación basada en pistas.
### Monitoreo

revisión de la propia comprensión.
### Relectura

volver a una parte para aclararla.

## Explicación completa para estudiante

Antes de leer, revisa título, imágenes y tipo de texto; decide para qué leerás. Durante la lectura, detente al finalizar un apartado y pregúntate qué entendiste. Si una palabra o una relación no queda clara, usa el contexto, consulta una fuente o relee. Después, explica la idea principal, selecciona evidencia y relaciona el contenido con el propósito inicial. Un lector competente no comprende todo de inmediato: detecta el problema y aplica una estrategia.

## Ejemplo explicado

Si debes descubrir por qué disminuyó una población de aves, lees con ese propósito. Al encontrar “pérdida de humedales”, anotas que podría ser una causa, continúas para buscar evidencia y finalmente resumes la relación.

## Procedimiento o estrategia recomendada

1. Lee o identifica la situación.
2. Reconoce la información importante.
3. Aplica la idea del tema.
4. Comprueba y explica tu respuesta.

## Actividad breve de 5 a 10 minutos

Lee un texto de tres párrafos. Escribe una pregunta antes, una duda durante y un resumen de dos oraciones después.

### Materiales

- Cuaderno o una hoja.
- Lápiz.

### Pasos

1. Lee la consigna completa.
2. Realiza la actividad.
3. Compara el resultado con la explicación.
4. Escribe una conclusión de una oración.

## Preguntas de práctica

1. ¿Qué diferencia hay entre predecir y adivinar?
2. ¿Qué haces si una oración contradice tu predicción?
3. ¿Cuándo conviene releer?
4. ¿Para qué sirve recordar el propósito?

## Respuestas esperadas y explicación

1. La predicción usa pistas del texto y conocimientos previos; la adivinanza puede no tener fundamento.
2. Se modifica la predicción usando la nueva información.
3. Cuando una idea, palabra o relación no se comprende.
4. Ayuda a seleccionar la información relevante.

## Errores comunes

- Seguir leyendo aunque no se comprenda nada.
- Mantener una predicción aun cuando el texto la contradice.
- Subrayar casi todo.

## Cómo explicarlo si todavía no se entiende

Comprender es construir significado mientras se lee. Para lograrlo se activa lo que ya se sabe, se formulan preguntas, se aclaran palabras, se relee y se resume. Trabaja primero con un ejemplo concreto y después vuelve a la definición.

## Adaptación TEA y lectura fácil

- Anticipar el objetivo y la cantidad de pasos.
- Presentar una idea principal por bloque y mantener visibles las palabras clave.
- Usar lenguaje literal; explicar expresiones figuradas o ambiguas.
- Ofrecer un ejemplo resuelto antes de la práctica.
- Reducir estímulos irrelevantes y permitir pausas sin penalización.
- No infantilizar ni asumir que una dificultad de comunicación implica menor comprensión.

### Versión en tres pasos

1. **Mira la idea central:** Comprender es construir significado mientras se lee. Para lograrlo se activa lo que ya se sabe, se formulan preguntas, se aclaran palabras, se relee y se resume.
2. **Observa un ejemplo:** revisa el ejemplo explicado de este archivo.
3. **Prueba una vez:** responde la primera pregunta y comprueba la respuesta.

## Para apoderados

Pida al estudiante que explique la idea con sus propias palabras. Ayude con una pregunta a la vez y valore el procedimiento, aunque la primera respuesta no sea correcta.

## Para docentes

- Comenzar activando conocimientos previos con una pregunta concreta.
- Modelar el procedimiento o razonamiento antes de solicitar trabajo independiente.
- Comprobar comprensión mediante una pregunta breve, no solo con “¿entendiste?”.
- Permitir distintas formas de respuesta: oral, escrita, esquema, tabla o selección.
- Cerrar con una síntesis y una conexión con el siguiente aprendizaje.

Conviene alternar explicación, modelado, práctica guiada y aplicación en un contexto nuevo.
