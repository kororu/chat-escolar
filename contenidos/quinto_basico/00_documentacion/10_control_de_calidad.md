# Lista de control de calidad

## Currículum
- [ ] Curso, asignatura y eje correctos.
- [ ] OA comprobado en fuente oficial.
- [ ] Fecha de verificación vigente.
- [ ] No se confunde propuesta curricular con normativa.

## Contenido
- [ ] Definiciones correctas y apropiadas para el nivel.
- [ ] Ejemplo resuelto coherente.
- [ ] Preguntas y respuestas revisadas.
- [ ] Errores frecuentes explicados sin ridiculizar.
- [ ] No se copió material protegido.

## Inclusión
- [ ] Lenguaje literal cuando se necesita.
- [ ] Pasos visibles.
- [ ] Opciones de participación.
- [ ] No infantiliza ni diagnostica.
- [ ] Ajustes no eliminan la habilidad central sin razón.

## Seguridad
- [ ] Experimentos seguros y supervisados.
- [ ] Sin asesoría médica o legal individual.
- [ ] Sin instrucciones de emergencia inventadas.
- [ ] Privacidad protegida.

## Técnica
- [ ] Front matter válido.
- [ ] ID único.
- [ ] Enlaces relativos correctos.
- [ ] UTF-8.
- [ ] Reindexación registrada.
