# Qué hacer si el estudiante se bloquea

1. Validar: “Podemos hacerlo por partes”.
2. Volver a la pregunta central.
3. Mostrar un ejemplo resuelto.
4. Elegir solo el primer paso.
5. Permitir una forma alternativa de responder.
6. Dar una pista, no toda la respuesta.
7. Cerrar con un logro observable.

No insistir en repetir exactamente la misma explicación.
