# Tema: Comprender y seguir instrucciones

## Respuesta breve

Una instrucción indica una acción que debe realizarse y puede tener un orden.

## Qué aprenderás

- Reconocer y explicar comprender y seguir instrucciones.
- Usar lo aprendido en una situación cercana.
- Comunicar una respuesta mediante palabras, dibujos, objetos o acciones.

## Objetivos de Aprendizaje relacionados

- LE01 OA 10
- LE01 OA 18

## Ideas fundamentales

- Una instrucción indica una acción que debe realizarse y puede tener un orden.
- El aprendizaje se construye con ejemplos cercanos y práctica breve.
- El estudiante puede demostrar comprensión de más de una forma.
- La retroalimentación debe indicar el paso específico que se debe revisar.

## Conceptos importantes

### Instrucción

indicación para realizar una acción.

### Paso

cada acción dentro de una secuencia.

### Orden

posición en que se hacen los pasos.

### Material

objeto necesario para una actividad.

## Explicación completa para estudiante

En primero básico se aprende a leer, escribir, escuchar y hablar de manera progresiva. Las palabras primero, luego y finalmente ayudan a seguir secuencias. Antes de actuar se escucha o lee completa la instrucción y se verifica si se necesitan materiales.

Para trabajar este tema conviene avanzar desde lo oral y concreto hacia lo escrito. Primero se escucha, observa o conversa; después se identifica la parte importante; finalmente se lee, escribe o explica. El objetivo no es memorizar una respuesta aislada, sino comprender cómo funciona el lenguaje para comunicar un mensaje.

Una respuesta correcta puede ser breve. Lo importante es que corresponda a la pregunta y que pueda relacionarse con una palabra, una oración, una ilustración o una parte del texto. Cuando aparece una palabra nueva, se explica con un ejemplo cercano y luego se vuelve a usar en otra situación.

La práctica debe ser frecuente y corta. Si el estudiante se equivoca, se revisa solo el paso necesario: sonido, sílaba, palabra, sentido de la oración o información del texto. Una instrucción indica una acción que debe realizarse y puede tener un orden.

## Ejemplo explicado

Para plantar una semilla: primero llena el recipiente con tierra, después coloca la semilla y finalmente agrega un poco de agua.

## Estrategia paso a paso

1. Escucha o lee la consigna.
2. Busca la palabra, sonido o información importante.
3. Responde con una palabra, oración, elección o dibujo.
4. Revisa si la respuesta coincide con la pregunta.

## Actividad breve de 5 a 10 minutos

Da instrucciones de tres pasos para ordenar materiales o preparar una colación simple sin usar objetos peligrosos.

### Materiales

- Hoja o cuaderno.
- Lápiz.
- Objetos cotidianos solo cuando la actividad los necesite.

### Pasos

1. Escucha o lee una instrucción.
2. Realiza un paso.
3. Comprueba lo que hiciste.
4. Explica o muestra tu respuesta.

## Preguntas de práctica

1. ¿Qué hace una instrucción?
2. ¿Por qué importa el orden?
3. ¿Qué palabra puede iniciar?
4. ¿Qué haces antes de comenzar?

## Respuestas esperadas y explicación

1. **Respuesta:** Indica una acción.
2. **Respuesta:** Porque cambiarlo puede alterar el resultado.
3. **Respuesta:** Primero.
4. **Respuesta:** Escuchas o lees la instrucción completa.

## Errores comunes

- Hacer solo el último paso.
- Unir varias instrucciones en una frase confusa.
- Usar procedimientos inseguros.

## Cómo explicarlo si todavía no se entiende

**Idea principal:** Una instrucción indica una acción que debe realizarse y puede tener un orden.

1. Mira un solo ejemplo.
2. Haz la actividad con objetos, dibujos o palabras.
3. Responde la primera pregunta.
4. Revisa la respuesta junto a un adulto o con Chat Escolar.

## Adaptación TEA y lectura fácil

- Anticipar qué se hará, cuántos pasos tendrá y cuándo terminará.
- Presentar una sola instrucción principal por vez.
- Usar lenguaje literal y explicar expresiones que puedan tener más de un sentido.
- Mantener visibles las palabras, números o imágenes que se están trabajando.
- Ofrecer un ejemplo terminado antes de pedir una respuesta independiente.
- Permitir señalar, ordenar, elegir, dibujar, escribir o responder oralmente.
- Reducir ruido visual y evitar animaciones, sonidos o recompensas inesperadas.
- Permitir pausas, repetición y tiempo adicional sin interpretar la demora como falta de interés.
- No infantilizar ni usar frases como “esto es fácil”.

### Versión en tres pasos

1. **Mira:** Una instrucción indica una acción que debe realizarse y puede tener un orden.
2. **Prueba:** realiza una parte del ejemplo.
3. **Comprueba:** compara con la primera respuesta esperada.

## Para apoderados

Trabaje entre 5 y 10 minutos. Formule una pregunta por vez y permita que el estudiante señale, manipule, dibuje o responda oralmente. No entregue toda la solución de inmediato: modele un ejemplo y acompañe el siguiente. Evite comparar su ritmo con el de otros niños.

## Para docentes

- Activar conocimientos previos con una situación concreta.
- Modelar antes de pedir trabajo independiente.
- Presentar pocos elementos por pantalla o página.
- Comprobar con una pregunta específica.
- Registrar qué apoyo permitió comprender.
- Cerrar con una idea principal y una experiencia de éxito.

## Cómo debería responder Chat Escolar

- Responder primero en una a tres oraciones.
- Usar vocabulario cotidiano y definir la palabra nueva.
- Dar una instrucción principal por vez.
- Mostrar un ejemplo concreto antes de la práctica.
- Ofrecer una pista antes de revelar una respuesta.
- No infantilizar ni decir que el ejercicio es fácil.
- Permitir los modos breve, lectura fácil, paso a paso y desafío.
