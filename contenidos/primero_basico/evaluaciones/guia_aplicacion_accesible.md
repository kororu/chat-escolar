# Guía de aplicación de evaluaciones — 1° básico

## Antes
- Explicar qué se hará y cuánto durará.
- Mostrar cómo responder.
- Preparar apoyos permitidos.
- Evitar aplicar cuando el estudiante está saturado.

## Durante
- Presentar pocos ítems.
- Diferenciar lectura de la consigna y habilidad evaluada.
- Registrar si se usó audio, objeto, dibujo, selección o tiempo adicional.
- No corregir con expresiones de decepción.
- Permitir pausa.

## Después
- Identificar habilidades logradas y el siguiente paso.
- No usar un resultado único para definir capacidad.
- Comparar respuestas con el tipo de apoyo.
- Compartir retroalimentación concreta.
