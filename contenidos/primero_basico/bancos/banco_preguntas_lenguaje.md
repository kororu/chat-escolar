# Banco de preguntas — Lenguaje y Comunicación — 1° básico

> Las preguntas deben presentarse de una en una en la interfaz.

## 01. Los textos transmiten mensajes y tienen propósito

1. ¿Qué hace un texto?
   - **Respuesta explicada:** Comunica un mensaje.
2. ¿Quién crea un texto?
   - **Respuesta explicada:** Un autor o autora, que puede ser una persona o un grupo.
3. ¿Para qué sirve una invitación?
   - **Respuesta explicada:** Para comunicar que alguien está invitado a una actividad.
4. ¿Todos los textos tienen el mismo propósito?
   - **Respuesta explicada:** No; pueden informar, entretener, pedir, enseñar o recordar.

## 02. Las palabras están separadas por espacios

1. ¿Cuántas palabras hay en «La rana salta»?
   - **Respuesta explicada:** Tres.
2. ¿Para qué sirven los espacios?
   - **Respuesta explicada:** Para separar las palabras y facilitar la lectura.
3. ¿Una palabra puede tener varias letras?
   - **Respuesta explicada:** Sí.
4. ¿Contar palabras es contar sílabas?
   - **Respuesta explicada:** No; son unidades distintas.

## 03. Separar y combinar sílabas

1. ¿Cómo se separa «mesa»?
   - **Respuesta explicada:** Me-sa.
2. ¿Qué palabra forman «so» y «pa»?
   - **Respuesta explicada:** Sopa.
3. ¿Todas las palabras tienen dos sílabas?
   - **Respuesta explicada:** No.
4. ¿Qué es combinar sílabas?
   - **Respuesta explicada:** Unirlas para formar una palabra.

## 04. Reconocer sonidos dentro de las palabras

1. ¿Con qué sonido comienza «mesa»?
   - **Respuesta explicada:** Con /m/.
2. ¿Qué palabra forman /p/ /a/ /n/?
   - **Respuesta explicada:** Pan.
3. ¿«Sol» y «sapo» comienzan igual?
   - **Respuesta explicada:** Sí, comienzan con /s/.
4. ¿Un fonema es una letra escrita?
   - **Respuesta explicada:** Es un sonido; una letra puede representarlo.

## 05. Relacionar letras y sonidos

1. ¿Qué representan las letras?
   - **Respuesta explicada:** Sonidos del habla.
2. ¿M y m son letras diferentes?
   - **Respuesta explicada:** Son dos formas de la misma letra.
3. ¿Qué sonido inicia «luna»?
   - **Respuesta explicada:** /l/.
4. ¿Para qué sirve relacionar letra y sonido?
   - **Respuesta explicada:** Para leer y escribir palabras.

## 06. Leer sílabas directas

1. ¿Qué tipo de sílaba es «pa»?
   - **Respuesta explicada:** Directa.
2. ¿Qué palabra forman «me» y «sa»?
   - **Respuesta explicada:** Mesa.
3. ¿Qué vocal aparece en «tu»?
   - **Respuesta explicada:** La u.
4. ¿Conviene separar siempre letra por letra?
   - **Respuesta explicada:** No; se busca reconocer la sílaba y luego la palabra.

## 07. Leer sílabas indirectas y complejas

1. ¿Qué sílaba inicial tiene «tren»?
   - **Respuesta explicada:** Tren; incluye el grupo tr.
2. ¿Por qué «plato» no es «pelato»?
   - **Respuesta explicada:** Porque no se agrega una vocal que no está escrita.
3. ¿Qué tipo de sílaba es «al»?
   - **Respuesta explicada:** Indirecta.
4. ¿Qué ayuda a leer una sílaba compleja?
   - **Respuesta explicada:** Unir los sonidos lentamente sin agregar otros.

## 08. Leer dígrafos rr, ll, ch y qu

1. ¿Qué combinación aparece en «chocolate»?
   - **Respuesta explicada:** Ch.
2. ¿Cómo se lee el inicio de «queso»?
   - **Respuesta explicada:** /ke/.
3. ¿Qué diferencia hay entre «pero» y «perro»?
   - **Respuesta explicada:** El sonido de r cambia y también puede cambiar el significado.
4. ¿Un dígrafo usa cuántas letras?
   - **Respuesta explicada:** Dos letras.

## 09. Leer palabras aisladas y en oraciones

1. ¿Qué ayuda a confirmar una palabra?
   - **Respuesta explicada:** Sus letras, sonidos y el contexto.
2. ¿Es correcto adivinar solo por el dibujo?
   - **Respuesta explicada:** No.
3. ¿Qué es autocorregirse?
   - **Respuesta explicada:** Notar un error y volver a leer correctamente.
4. ¿Por qué importa el sentido?
   - **Respuesta explicada:** Porque la oración debe comunicar una idea coherente.

## 10. Leer en voz alta con precisión

1. ¿Qué significa leer con precisión?
   - **Respuesta explicada:** Pronunciar lo que está escrito sin cambios.
2. ¿Es malo autocorregirse?
   - **Respuesta explicada:** No; muestra que se revisó la lectura.
3. ¿Se debe leer muy rápido?
   - **Respuesta explicada:** No; primero importa precisión y comprensión.
4. ¿Qué conviene practicar?
   - **Respuesta explicada:** Textos breves y conocidos.

## 11. Respetar puntos y pausas al leer

1. ¿Qué indica un punto?
   - **Respuesta explicada:** Que termina una oración.
2. ¿Se continúa sin pausa después del punto?
   - **Respuesta explicada:** No.
3. ¿Qué comienza después del punto?
   - **Respuesta explicada:** Una nueva oración.
4. ¿Las pausas ayudan a comprender?
   - **Respuesta explicada:** Sí.

## 12. Relacionar el texto con experiencias y conocimientos

1. ¿Para qué sirve una conexión?
   - **Respuesta explicada:** Para comprender mejor.
2. ¿La experiencia personal reemplaza el texto?
   - **Respuesta explicada:** No.
3. ¿Qué debes revisar para responder?
   - **Respuesta explicada:** La información del texto.
4. ¿Todos tendrán la misma conexión?
   - **Respuesta explicada:** No, porque sus experiencias pueden ser distintas.

## 13. Visualizar lo que describe un texto

1. ¿Qué es visualizar?
   - **Respuesta explicada:** Formar una imagen mental.
2. ¿Qué guía la imagen?
   - **Respuesta explicada:** Las palabras y detalles del texto.
3. ¿Todos dibujarán igual?
   - **Respuesta explicada:** No necesariamente.
4. ¿Puede el dibujo contradecir un detalle claro?
   - **Respuesta explicada:** No debería.

## 14. Conocer cuentos, poemas y textos predecibles

1. ¿Por qué leer textos variados?
   - **Respuesta explicada:** Para disfrutar, imaginar y aprender.
2. ¿Qué puede repetirse en un texto predecible?
   - **Respuesta explicada:** Una frase, acción o estructura.
3. ¿Un poema siempre cuenta una historia?
   - **Respuesta explicada:** No.
4. ¿Releer ayuda?
   - **Respuesta explicada:** Sí, mejora reconocimiento y comprensión.

## 15. Encontrar información explícita en narraciones

1. ¿Dónde aparece la información explícita?
   - **Respuesta explicada:** Directamente en el texto o narración.
2. Si dice «Lina abrió la caja», ¿quién abrió?
   - **Respuesta explicada:** Lina.
3. ¿Debes inventar para responder?
   - **Respuesta explicada:** No.
4. ¿Qué ayuda a encontrar un dato?
   - **Respuesta explicada:** Volver a la oración relacionada.

## 16. Comprender información implícita sencilla

1. ¿Qué necesitas para inferir?
   - **Respuesta explicada:** Pistas y conocimientos.
2. ¿La inferencia puede ser cualquier idea?
   - **Respuesta explicada:** No.
3. ¿Qué indica un paraguas mojado?
   - **Respuesta explicada:** Probablemente que llueve o llovió.
4. ¿Cómo justificas?
   - **Respuesta explicada:** Nombrando la pista.

## 17. Responder qué, quién, dónde, cuándo y por qué

1. ¿Qué pregunta busca un lugar?
   - **Respuesta explicada:** Dónde.
2. ¿Qué pregunta busca una persona?
   - **Respuesta explicada:** Quién.
3. ¿Qué busca «por qué»?
   - **Respuesta explicada:** Una causa o razón.
4. ¿Por qué hay que escuchar la pregunta completa?
   - **Respuesta explicada:** Para responder el tipo de información solicitado.

## 18. Ordenar acciones y reconocer personajes y lugares

1. ¿Qué es una secuencia?
   - **Respuesta explicada:** El orden de los hechos.
2. ¿Quién participa en una historia?
   - **Respuesta explicada:** Los personajes.
3. ¿Qué palabra introduce el último hecho?
   - **Respuesta explicada:** Al final.
4. ¿Puede una historia tener varios lugares?
   - **Respuesta explicada:** Sí.

## 19. Relacionar ilustraciones con la historia

1. ¿Qué puede aportar una ilustración?
   - **Respuesta explicada:** Detalles visuales sobre personajes, lugares o acciones.
2. ¿La imagen reemplaza las palabras?
   - **Respuesta explicada:** No.
3. ¿Qué significa complementar?
   - **Respuesta explicada:** Agregar información relacionada.
4. ¿Qué haces si texto e imagen parecen distintos?
   - **Respuesta explicada:** Vuelves a observar y explicas la diferencia.

## 20. Dar una opinión sobre una lectura

1. ¿Todas las opiniones deben ser iguales?
   - **Respuesta explicada:** No.
2. ¿Qué fortalece una opinión?
   - **Respuesta explicada:** Una razón relacionada con el texto.
3. ¿«Me gustó» es una opinión?
   - **Respuesta explicada:** Sí, pero puede explicarse mejor con una razón.
4. ¿Se debe respetar una opinión diferente?
   - **Respuesta explicada:** Sí.

## 21. Poemas, rimas y tradición oral

1. ¿Qué palabras riman: gato y pato?
   - **Respuesta explicada:** Sí.
2. ¿Qué es un verso?
   - **Respuesta explicada:** Una línea de un poema.
3. ¿Una rima debe tener exactamente las mismas letras?
   - **Respuesta explicada:** No; importa el sonido final.
4. ¿Qué es tradición oral?
   - **Respuesta explicada:** Textos que se transmiten al decirlos y escucharlos.

## 22. Comprender cartas y notas

1. ¿Quién es el destinatario?
   - **Respuesta explicada:** La persona que recibe el mensaje.
2. ¿Para qué sirve una firma?
   - **Respuesta explicada:** Para saber quién escribió.
3. ¿Una nota suele ser breve?
   - **Respuesta explicada:** Sí.
4. ¿Qué debe comunicar con claridad?
   - **Respuesta explicada:** El mensaje principal.

## 23. Comprender y seguir instrucciones

1. ¿Qué hace una instrucción?
   - **Respuesta explicada:** Indica una acción.
2. ¿Por qué importa el orden?
   - **Respuesta explicada:** Porque cambiarlo puede alterar el resultado.
3. ¿Qué palabra puede iniciar?
   - **Respuesta explicada:** Primero.
4. ¿Qué haces antes de comenzar?
   - **Respuesta explicada:** Escuchas o lees la instrucción completa.

## 24. Comprender artículos informativos simples

1. ¿Qué hace un artículo informativo?
   - **Respuesta explicada:** Entrega datos o explicaciones.
2. ¿Qué anticipa el título?
   - **Respuesta explicada:** El tema.
3. ¿Un dato es lo mismo que una preferencia?
   - **Respuesta explicada:** No.
4. ¿Qué puedes preguntar después de leer?
   - **Respuesta explicada:** Qué aprendí y qué duda me queda.

## 25. Explorar libros y usar la biblioteca

1. ¿Qué puedes mirar para elegir un libro?
   - **Respuesta explicada:** Portada, título, tema e ilustraciones.
2. ¿Por qué se cuidan los libros?
   - **Respuesta explicada:** Porque son compartidos y deben durar.
3. ¿Qué significa préstamo?
   - **Respuesta explicada:** Usar temporalmente y devolver.
4. ¿Se puede releer un libro?
   - **Respuesta explicada:** Sí.

## 26. Escribir para comunicar ideas, hechos y sentimientos

1. ¿Para qué sirve escribir?
   - **Respuesta explicada:** Para comunicar.
2. ¿Se puede planificar hablando primero?
   - **Respuesta explicada:** Sí.
3. ¿Qué diferencia hay entre hecho y sentimiento?
   - **Respuesta explicada:** El hecho cuenta qué ocurrió; el sentimiento expresa cómo se vivió.
4. ¿El primer intento debe ser perfecto?
   - **Respuesta explicada:** No; puede revisarse.

## 27. Escribir oraciones completas

1. ¿Qué debe comunicar una oración?
   - **Respuesta explicada:** Una idea completa.
2. ¿«El perro corre» es una oración?
   - **Respuesta explicada:** Sí.
3. ¿Pueden cambiarse palabras?
   - **Respuesta explicada:** Sí, si la oración conserva sentido.
4. ¿Los espacios son necesarios?
   - **Respuesta explicada:** Sí.

## 28. Escribir con letra clara y separar palabras

1. ¿Para qué sirve una letra clara?
   - **Respuesta explicada:** Para que otros puedan leer.
2. ¿Qué separa palabras?
   - **Respuesta explicada:** Un espacio.
3. ¿Todas las letras deben verse idénticas?
   - **Respuesta explicada:** No, pero deben poder reconocerse.
4. ¿Se pueden usar apoyos?
   - **Respuesta explicada:** Sí.

## 29. Usar vocabulario nuevo al escribir

1. ¿De dónde pueden venir palabras nuevas?
   - **Respuesta explicada:** De textos leídos o escuchados.
2. ¿Se usa una palabra sin conocerla?
   - **Respuesta explicada:** No conviene.
3. ¿Qué ayuda a comprobar el uso?
   - **Respuesta explicada:** Leer la oración y revisar el contexto.
4. ¿Una palabra precisa mejora el mensaje?
   - **Respuesta explicada:** Sí.

## 30. Disfrutar cuentos, poemas, fábulas y leyendas escuchados

1. ¿Por qué escuchar textos que aún no puedes leer solo?
   - **Respuesta explicada:** Porque permiten acceder a historias y vocabulario más amplios.
2. ¿Qué puedes hacer mientras escuchas?
   - **Respuesta explicada:** Imaginar, formular preguntas y seguir la historia.
3. ¿Una fábula es igual a una noticia?
   - **Respuesta explicada:** No.
4. ¿Se debe detener cada palabra?
   - **Respuesta explicada:** No; se pueden seleccionar algunas para explicar.

## 31. Comprender explicaciones, relatos e instrucciones orales

1. ¿Qué haces si no entiendes?
   - **Respuesta explicada:** Pides aclaración o repetición.
2. ¿Cuántas acciones hay en «abre y subraya»?
   - **Respuesta explicada:** Dos.
3. ¿Escuchar significa quedarse inmóvil?
   - **Respuesta explicada:** No necesariamente.
4. ¿Qué ayuda a recordar?
   - **Respuesta explicada:** Repetir con palabras propias o usar apoyo visual.

## 32. Averiguar palabras y expresiones desconocidas

1. ¿Qué puedes hacer si no conoces una palabra?
   - **Respuesta explicada:** Preguntar o buscar pistas.
2. ¿Qué es una definición?
   - **Respuesta explicada:** Una explicación del significado.
3. ¿El contexto siempre basta?
   - **Respuesta explicada:** No, a veces se necesita ayuda.
4. ¿Por qué conviene usar la palabra?
   - **Respuesta explicada:** Para recordarla y comprenderla mejor.

   - **Respuesta explicada:** Explorar historias, personajes y expresión.
2. ¿Todos deben actuar de la misma forma?
   - **Respuesta explicada:** No.
3. ¿Qué es un personaje?
   - **Respuesta explicada:** El rol representado.
4. ¿Se pueden usar títeres?
   - **Respuesta explicada:** Sí.

## 34. Conversar, escuchar y respetar turnos

1. ¿Qué debes hacer mientras otra persona habla?
   - **Respuesta explicada:** Escuchar y esperar el turno.
2. ¿Puedes hacer una pregunta?
   - **Respuesta explicada:** Sí.
3. ¿Qué significa mantener el tema?
   - **Respuesta explicada:** Hablar de asuntos relacionados.
4. ¿Participar siempre significa hablar mucho?
   - **Respuesta explicada:** No.

## 35. Saludos, cortesía y expresión de necesidades

1. ¿Qué palabra puede acompañar una petición?
   - **Respuesta explicada:** Por favor.
2. ¿Cuándo se usa «permiso»?
   - **Respuesta explicada:** Cuando se necesita pasar o realizar una acción que requiere autorización.
3. ¿Se debe obligar a abrazar al saludar?
   - **Respuesta explicada:** No.
4. ¿Cómo expresas que necesitas ayuda?
   - **Respuesta explicada:** Lo dices de forma directa y respetuosa.

## 36. Hablar de un tema con orden, descripción y volumen adecuado

1. ¿Qué necesita una presentación breve?

2. ¿Volumen adecuado significa gritar?
   - **Respuesta explicada:** No.

   - **Respuesta explicada:** Palabras clave o dibujos.
4. ¿Se puede usar vocabulario nuevo?

2. ¿Por qué se aclaran los roles?

3. ¿Todos deben hacer exactamente lo mismo?
   - **Respuesta explicada:** No.
4. ¿Qué pasa si alguien necesita cambiar de rol?
   - **Respuesta explicada:** Se conversa y se ajusta.

   - **Respuesta explicada:** Sí.

   - **Respuesta explicada:** La entonación y las pausas.
