# Guía para familias

- Trabaje en períodos breves.
- Pregunte una cosa por vez.
- Modele un ejemplo y acompañe el siguiente.
- Lea en voz alta sin convertir cada página en una prueba.
- Repita textos favoritos.
- Use objetos cotidianos para cantidades, pero cuide piezas pequeñas.
- Valore autocorrección y esfuerzo.
- No compare con hermanos o compañeros.
- No fuerce contacto visual ni respuesta inmediata.
- Coordine con la escuela si una barrera se mantiene.
- Consulte fuentes oficiales ante información curricular, institucional o de seguridad que pueda cambiar.
