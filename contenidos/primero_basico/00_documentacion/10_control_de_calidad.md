# Control de calidad — 1° básico

- [ ] Curso, asignatura y eje correctos.
- [ ] OA verificado o marcado para revisión.
- [ ] Texto original.
- [ ] Respuesta breve comprensible sin ayuda adicional.
- [ ] Una palabra nueva definida.
- [ ] Ejemplo apropiado para 6 a 7 años.
- [ ] Actividad segura y realizable.
- [ ] Una instrucción principal por paso.
- [ ] Preguntas y respuestas revisadas.
- [ ] Números, secuencias y representaciones comprobados.
- [ ] Adaptación TEA sin infantilización.
- [ ] Privacidad protegida.
- [ ] No hay estereotipos de familia, género, cultura o discapacidad.
- [ ] Metadatos y enlaces válidos.
- [ ] Fecha y versión registradas.
