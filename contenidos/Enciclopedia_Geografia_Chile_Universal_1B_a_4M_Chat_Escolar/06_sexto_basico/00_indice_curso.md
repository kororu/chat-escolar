# Índice de Geografía — 6° básico

> **Nota curricular 2026:** esta enciclopedia fue organizada el 2026-07-14 tomando como referencia el portal oficial Currículum Nacional/MINEDUC. Para 1° a 6° básico, la geografía se integra en Historia, Geografía y Ciencias Sociales, principalmente mediante ubicación, mapas, paisajes, América, Chile, recursos y zonas naturales. En 7° básico a 2° medio continúa como parte de Historia, Geografía y Ciencias Sociales. En 3° y 4° medio se diferencia entre Formación General y asignaturas/electivos como Geografía, Territorio y Desafíos Socioambientales. Cuando un tema se usa como ampliación o puente curricular, se marca explícitamente y no se presenta como OA obligatorio.


## Propósito

Este índice organiza los contenidos de Geografía de Chile y Geografía Universal para **6° básico**. Puede usarse como guía de navegación para estudiantes, apoderados, docentes y para el buscador local de Chat Escolar.

## Características del lenguaje recomendado

- Usar explicaciones claras y precisas.
- Partir desde lugares concretos y avanzar hacia mapas, datos o conceptos.
- Separar ubicación, características y relación sociedad-espacio.
- Incluir ejemplos cotidianos.
- Para estudiantes con TEA, usar instrucciones de un paso, anticipación, tablas y esquemas.

## Habilidades principales

- Observar el espacio.
- Ubicar lugares en mapas, planos o globos.
- Describir paisajes y territorios.
- Comparar lugares.
- Leer simbología, escala y leyenda.
- Relacionar sociedad y ambiente.
- Interpretar datos espaciales.
- Proponer medidas de cuidado, prevención o planificación.

## Preguntas guía

1. ¿Dónde ocurre?
2. ¿Cómo es el lugar?
3. ¿Qué elementos naturales y humanos hay?
4. ¿Qué relaciones se observan?
5. ¿Qué información falta verificar?
6. ¿Cómo se puede representar en un mapa?
