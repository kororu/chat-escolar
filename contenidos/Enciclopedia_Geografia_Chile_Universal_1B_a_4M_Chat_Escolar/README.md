# Enciclopedia de Geografía de Chile y Geografía Universal — Chat Escolar

> **Nota curricular 2026:** esta enciclopedia fue organizada el 2026-07-14 tomando como referencia el portal oficial Currículum Nacional/MINEDUC. Para 1° a 6° básico, la geografía se integra en Historia, Geografía y Ciencias Sociales, principalmente mediante ubicación, mapas, paisajes, América, Chile, recursos y zonas naturales. En 7° básico a 2° medio continúa como parte de Historia, Geografía y Ciencias Sociales. En 3° y 4° medio se diferencia entre Formación General y asignaturas/electivos como Geografía, Territorio y Desafíos Socioambientales. Cuando un tema se usa como ampliación o puente curricular, se marca explícitamente y no se presenta como OA obligatorio.


## Objetivo

Crear una base local en Markdown para que Chat Escolar pueda responder preguntas de geografía desde 1° básico hasta 4° medio, adaptando la respuesta al curso, perfil activo y nivel de comprensión.

## Qué incluye

- Geografía de Chile.
- Geografía Universal.
- Mapas, planos, globos, coordenadas, leyendas y escalas.
- Continentes y océanos.
- Relieve, clima, hidrografía y regiones naturales.
- Zonas naturales y regiones de Chile.
- Recursos naturales y desarrollo sostenible.
- Población, migraciones, ciudades y ruralidad.
- Riesgos naturales y desastres socionaturales.
- Territorio, paisaje, lugar, cartografía y sociedad-espacio.
- Cambio climático, justicia ambiental y geopolítica.
- Educación ciudadana territorial y geografía avanzada de 3° y 4° medio.

## Uso recomendado

Instalar como:

```text
contenidos/geografia_chile_universal/
```

No reemplaza las carpetas por curso. Funciona como colección transversal especializada.

## Estructura

```text
geografia_chile_universal/
├── 00_documentacion/
├── 01_primero_basico/
├── ...
├── 12_cuarto_medio/

├── 14_bancos_generales/
├── 15_evaluaciones_generales/
├── 16_apoyo_inclusivo/
├── 17_compendios/
└── 18_rag/
```

## Prioridad de recuperación sugerida

1. Contenido curricular del curso activo.
2. Geografía del mismo curso.
3. Tema transversal relacionado.
4. Curso anterior para prerrequisitos.
5. Curso superior solo si se marca como profundización.

## Regla clave

Si una pregunta requiere datos actuales —población, límites, autoridades, cifras de migración, clima reciente, desastres, indicadores sociales— Chat Escolar debe buscar fuente verificada o indicar que la cifra necesita actualización.
