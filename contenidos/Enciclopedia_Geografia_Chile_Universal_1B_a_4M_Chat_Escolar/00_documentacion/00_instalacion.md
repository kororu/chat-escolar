# Instalación en Chat Escolar

1. Extrae el ZIP.
2. Renombra la carpeta `Enciclopedia_Geografia_Chile_Universal_1B_a_4M_Chat_Escolar` como `geografia_chile_universal`.
3. Mueve la carpeta a:

```text
C:\Users\Ariel\Documents\proyectos\chat-escolar\contenidos\geografia_chile_universal
```

## Estructura esperada

```text
chat-escolar/
└── contenidos/
    ├── primero_basico/
    ├── segundo_basico/
    ├── ...
    ├── historia_chile/
    ├── historia_universal/
    └── geografia_chile_universal/
```

## No indexar inicialmente

```text
geografia_chile_universal/17_compendios/
```

Los compendios duplican información y pueden afectar la recuperación.
