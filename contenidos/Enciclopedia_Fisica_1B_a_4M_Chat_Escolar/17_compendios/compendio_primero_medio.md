# Qué es una onda

> **Referencia curricular:** colección organizada el 2026-07-14. Distingue contenidos curriculares directos, puentes y profundizaciones. Los datos tecnológicos, energéticos, astronómicos o normativos actuales requieren fuente verificada.


## Respuesta breve

**Qué es una onda** es un tema de Física que permite describir y explicar cambios en el movimiento, la energía, la materia, las ondas o el universo mediante mediciones, modelos y relaciones matemáticas.

## Explicación clara

La Física estudia patrones del mundo material. Para este tema conviene distinguir el sistema observado, las magnitudes que se miden, las interacciones presentes, el modelo utilizado y sus límites.

Una explicación física responde:

1. ¿Qué sistema se analiza?
2. ¿Qué magnitudes cambian?
3. ¿Qué interacción o transferencia ocurre?
4. ¿Qué evidencia se puede medir?
5. ¿Qué modelo representa el fenómeno?
6. ¿En qué condiciones deja de funcionar el modelo?

## Conceptos clave

- **Sistema:** parte del mundo seleccionada para estudiar.
- **Magnitud:** propiedad medible con unidad.
- **Fuerza:** interacción capaz de cambiar movimiento o deformar.
- **Energía:** magnitud asociada a cambios y transferencias.
- **Modelo:** representación simplificada.
- **Evidencia:** mediciones u observaciones que permiten evaluar una explicación.

## Ejemplo cotidiano

Una regla que sobresale del borde de una mesa vibra y produce un sonido; cambiar la longitud modifica la frecuencia.

El ejemplo ayuda a imaginar o medir el fenómeno. No sustituye la teoría completa. Toda analogía debe incluir sus límites.

**Sismos, tsunamis y medición** es un tema de Física que permite describir y explicar cambios en el movimiento, la energía, la materia, las ondas o el universo mediante mediciones, modelos y relaciones matemáticas.

## Explicación clara

La Física estudia patrones del mundo material. Para este tema conviene distinguir el sistema observado, las magnitudes que se miden, las interacciones presentes, el modelo utilizado y sus límites.

Una explicación física responde:

1. ¿Qué sistema se analiza?
2. ¿Qué magnitudes cambian?
3. ¿Qué interacción o transferencia ocurre?
4. ¿Qué evidencia se puede medir?
5. ¿Qué modelo representa el fenómeno?
6. ¿En qué condiciones deja de funcionar el modelo?

## Conceptos clave

- **Sistema:** parte del mundo seleccionada para estudiar.
- **Magnitud:** propiedad medible con unidad.
- **Fuerza:** interacción capaz de cambiar movimiento o deformar.
- **Energía:** magnitud asociada a cambios y transferencias.
- **Modelo:** representación simplificada.
- **Evidencia:** mediciones u observaciones que permiten evaluar una explicación.

## Ejemplo cotidiano

Observar un fenómeno cotidiano, medir una variable y comparar resultados ayuda a construir una explicación física.

El ejemplo ayuda a imaginar o medir el fenómeno. No sustituye la teoría completa. Toda analogía debe incluir sus límites.

- Una variable por vez.
- Instrucciones numeradas.
- Anticipar sonidos, luces o movimientos.
- Evitar estímulos intensos.
- Permitir responder con tabla, dibujo o selección.
- No obligar a tocar materiales o escuchar sonidos molestos.

## Actividad o experimento seguro

1. Formula una pregunta.
2. Define una variable que cambiará.
3. Mantén constantes las demás condiciones.
4. Usa materiales de bajo riesgo.
5. Mide al menos tres veces cuando sea posible.
6. Registra en tabla.
7. Calcula o compara.
8. Escribe una conclusión y una limitación.

**No usar:** enchufes domésticos abiertos, fuego sin protocolo, láseres dirigidos a ojos, sonidos intensos, recipientes presurizados, proyectiles, sustancias desconocidas o radiación ionizante.

## Preguntas de práctica

### Inicial
1. ¿Qué fenómeno se estudia?
2. ¿Qué magnitud puede observarse o medirse?
3. Da un ejemplo cotidiano.

### Intermedio
4. ¿Qué variable cambiarías?
5. ¿Qué unidad usarías?
6. ¿Qué fuerza o transferencia interviene?

### Desafío
7. Diseña una prueba segura.
8. Representa el resultado con un gráfico o ecuación.
9. Explica una limitación del modelo.
10. Relaciona el tema con tecnología, ambiente o sociedad.

## Errores frecuentes

- Confundir masa y peso.
- Confundir calor y temperatura.
- Creer que se necesita fuerza para mantener cualquier movimiento uniforme.
- Pensar que la energía desaparece.
- Confundir rapidez y velocidad.
- Tratar analogías como mecanismos literales.
- Ignorar unidades.
- Concluir causalidad sin controlar variables.
- Usar datos actuales sin fuente.
