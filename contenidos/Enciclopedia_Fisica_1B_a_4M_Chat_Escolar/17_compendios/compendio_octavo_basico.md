# Carga eléctrica

> **Referencia curricular:** colección organizada el 2026-07-14. Distingue contenidos curriculares directos, puentes y profundizaciones. Los datos tecnológicos, energéticos, astronómicos o normativos actuales requieren fuente verificada.


## Respuesta breve

**Carga eléctrica** es un tema de Física que permite describir y explicar cambios en el movimiento, la energía, la materia, las ondas o el universo mediante mediciones, modelos y relaciones matemáticas.

## Explicación clara

La Física estudia patrones del mundo material. Para este tema conviene distinguir el sistema observado, las magnitudes que se miden, las interacciones presentes, el modelo utilizado y sus límites.

Una explicación física responde:

1. ¿Qué sistema se analiza?
2. ¿Qué magnitudes cambian?
3. ¿Qué interacción o transferencia ocurre?
4. ¿Qué evidencia se puede medir?
5. ¿Qué modelo representa el fenómeno?
6. ¿En qué condiciones deja de funcionar el modelo?

## Conceptos clave

- **Sistema:** parte del mundo seleccionada para estudiar.
- **Magnitud:** propiedad medible con unidad.
- **Fuerza:** interacción capaz de cambiar movimiento o deformar.
- **Energía:** magnitud asociada a cambios y transferencias.
- **Modelo:** representación simplificada.
- **Evidencia:** mediciones u observaciones que permiten evaluar una explicación.

## Ejemplo cotidiano

Observar un fenómeno cotidiano, medir una variable y comparar resultados ayuda a construir una explicación física.

El ejemplo ayuda a imaginar o medir el fenómeno. No sustituye la teoría completa. Toda analogía debe incluir sus límites.
