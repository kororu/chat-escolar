5. Calcular paso a paso.
6. Comprobar con estimación u operación inversa.

## Conceptos clave

- **Número:** representa cantidad, orden o medida.
- **Operación:** acción matemática entre números.
- **Resultado:** valor obtenido.
- **Estimación:** aproximación razonable.
- **Operación inversa:** permite comprobar otra operación.
- **Unidad:** indica qué se cuenta o mide.

## Ejemplo cotidiano

Una temperatura de 3 °C bajo cero se representa con −3.

## Resolución paso a paso

```text
DATOS → PREGUNTA → OPERACIÓN → CÁLCULO → COMPROBACIÓN → RESPUESTA
```

Ejemplo: 4 cuadernos cuestan $1.500 cada uno.

2. Operación: `4 × 1.500`.
3. Cálculo: `6.000`.
4. Comprobación: `1.500 + 1.500 + 1.500 + 1.500 = 6.000`.
5. Respuesta: cuestan $6.000.
