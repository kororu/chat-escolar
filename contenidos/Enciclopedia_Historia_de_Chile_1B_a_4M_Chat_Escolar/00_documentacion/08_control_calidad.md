# Control de calidad

- [ ] Curso y tipo de alineación correctos.
- [ ] OA usado como referencia y no como cita inventada.
- [ ] Fecha y período coherentes.
- [ ] Actores diversos y no estereotipados.
- [ ] Causas múltiples cuando corresponda.
- [ ] Diferencia entre hecho e interpretación.
- [ ] Tema sensible con aviso y alternativas.
- [ ] Preguntas y respuestas concordantes.
- [ ] Texto original y sin reproducción extensa de fuentes.
- [ ] Enlaces internos y JSONL validados.
