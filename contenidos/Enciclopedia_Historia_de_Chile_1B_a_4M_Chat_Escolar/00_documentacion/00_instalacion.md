# Instalación en Chat Escolar

1. Extraer el ZIP.
2. Renombrar la carpeta, si se desea, como `historia_chile`.
3. Copiarla en `chat-escolar/contenidos/historia_chile/`.
4. Mantener las enciclopedias por curso existentes: esta base es transversal y complementaria.
5. Crear una colección vectorial separada para evitar que la misma pregunta recupere capítulos duplicados de las enciclopedias generales.
6. Probar primero las consultas de `00_documentacion/09_pruebas_recuperacion.md`.
