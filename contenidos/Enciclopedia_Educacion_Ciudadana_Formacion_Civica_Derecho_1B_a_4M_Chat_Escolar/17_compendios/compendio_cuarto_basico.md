# Derechos fundamentales y vida cotidiana

> **Estado de referencia:** colección organizada el 2026-07-14. Los capítulos jurídicos explican conceptos escolares generales. No sustituyen asesoría legal. Toda norma, autoridad, procedimiento, plazo o requisito vigente debe comprobarse en fuentes oficiales actualizadas, especialmente Biblioteca del Congreso Nacional, Poder Judicial, Congreso Nacional, municipalidades, Servel y organismos competentes.


## Respuesta breve

**Derechos fundamentales y vida cotidiana** se estudia para comprender cómo las personas conviven, ejercen derechos, cumplen responsabilidades, participan en decisiones y resuelven conflictos mediante reglas e instituciones.

## Explicación clara

La educación ciudadana no consiste solo en memorizar nombres de autoridades. Ayuda a entender tres relaciones:

1. **Persona y comunidad:** cada persona tiene dignidad, derechos y responsabilidades.
2. **Reglas e instituciones:** las normas organizan la convivencia y las instituciones cumplen funciones que deben estar limitadas por el derecho.
3. **Participación y bien común:** una democracia necesita personas informadas, capaces de dialogar, organizarse, votar cuando corresponda y controlar pacíficamente a las autoridades.

En **Derechos fundamentales y vida cotidiana**, se debe diferenciar entre una norma cotidiana, una regla escolar, una ley y una regla constitucional. También se debe distinguir entre lo que una institución **debería hacer según una explicación general** y lo que establece actualmente una norma específica.

## Conceptos clave

- **derechos fundamentales:** concepto necesario para comprender y buscar este tema.
- **igualdad:** concepto necesario para comprender y buscar este tema.
- **libertad:** concepto necesario para comprender y buscar este tema.
- **protección:** concepto necesario para comprender y buscar este tema.

## Ejemplo cotidiano

Imagina que un curso necesita decidir cómo usar un espacio común. Primero escucha distintas propuestas. Luego acuerda criterios: seguridad, acceso para todos y cuidado de materiales. Puede votar, pero también debe escuchar a quienes prefieren otra opción.

La votación permite decidir. El respeto a las reglas y a las personas permite que la decisión sea democrática.

## Ejemplo adaptado para estudiantes con TEA

Se presenta una secuencia predecible:

```text
1. ¿Qué ocurrió?
2. ¿Qué regla o derecho se relaciona?
3. ¿Quiénes participan?
4. ¿Qué opciones existen?
5. ¿Qué solución evita daño y respeta derechos?
```

Ejemplo:

> “Dos estudiantes quieren usar el mismo material. No se decide quién es mejor. Se puede usar un turno, compartir, buscar otro material o pedir mediación.”

El estudiante puede responder eligiendo una opción, ordenando pasos o completando una tabla. No es obligatorio relatar conflictos personales.

## Diferencias importantes

| Concepto | Explicación escolar |
|---|---|
| Norma | Regla que orienta una conducta en un grupo o situación. |
| Ley | Norma jurídica aprobada mediante un procedimiento oficial y obligatoria dentro de su ámbito. |
| Derecho | Facultad o protección reconocida a las personas. |
| Deber o responsabilidad | Conducta esperada para respetar derechos, reglas y bien común. |
| Institución | Organización con funciones definidas. |
| Justicia | Búsqueda de decisiones imparciales, justificadas y respetuosas de derechos. |

## Actividad breve

### Caso para analizar

Una comunidad escolar quiere disminuir los mensajes ofensivos en un grupo digital.

1. Identifica el problema.
2. Reconoce derechos involucrados.
3. Propón una regla clara.
4. Indica qué debe ocurrir si alguien necesita ayuda.
5. Explica cómo evitar una sanción humillante o desproporcionada.

## Preguntas de práctica

### Inicial

1. ¿Cuál es la idea principal del tema?
2. Nombra un derecho o responsabilidad relacionado.
3. ¿Qué institución, persona o grupo podría intervenir?

### Intermedio

4. ¿Qué diferencia hay entre una opinión y una regla?
5. ¿Cómo se puede resolver el conflicto sin violencia?
6. ¿Qué información necesitarías verificar?

### Desafío

7. ¿La solución propuesta respeta a la mayoría y a la minoría?
8. ¿Qué control debería existir sobre quien toma la decisión?
9. ¿Cómo cambiaría el caso en internet?

## Respuestas esperadas

- Debe identificar derechos, responsabilidades, reglas o instituciones pertinentes.
- Una solución democrática debe poder explicarse, admitir revisión y evitar discriminación.
- Si se habla de leyes, autoridades, competencias o procedimientos vigentes, debe consultarse una fuente oficial.
- Resolver pacíficamente no significa aceptar abuso: puede ser necesario pedir protección o recurrir a una institución.

## Errores comunes

- Creer que democracia significa que la mayoría puede hacer cualquier cosa.
- Confundir derecho con deseo.
- Pensar que toda regla es una ley.
- Suponer que una autoridad no tiene límites.
- Confundir justicia con venganza.
- Compartir datos personales para “demostrar” un problema.
- Dar consejos legales específicos sin revisar la legislación vigente.
- Presentar una institución de manera simplificada como si resolviera todos los casos.

## Cómo explicarlo si el estudiante no entiende

> “Vivir con otras personas requiere reglas. Las reglas deben proteger a las personas. Cuando hay un problema, hablamos, revisamos derechos y buscamos una solución justa.”

## Para apoderados

Use situaciones cotidianas: turnos, permisos, mensajes digitales, cuidado de espacios y decisiones familiares. Evite explicar ciudadanía únicamente como obediencia. Pregunte también quién participa, qué derecho se protege y cómo se revisa una decisión injusta.

## Para docentes

Trabaje con casos breves, roles claros y fuentes institucionales. Antes de un debate, entregue:

- conceptos;
- reglas de diálogo;
- evidencia mínima;
- opciones de participación;
- forma de cerrar y evaluar.

No obligue a estudiantes a representar violencia, discriminación o victimización.

## Mini resumen

- Todas las personas tienen dignidad.
- Los derechos necesitan protección y responsabilidades.
- Las instituciones tienen funciones y límites.
- La participación requiere información, respeto y control.
- Los datos jurídicos actuales deben verificarse.

## Palabras clave para búsqueda local

derechos fundamentales, igualdad, libertad, protección

## Temas relacionados

### Cursos anteriores

normas, respeto y convivencia

### Cursos posteriores

democracia, derechos, instituciones y participación

## Reglas para Chat Escolar

- Adaptar lenguaje y extensión al curso activo.
- Para estudiante: respuesta clara, ejemplo y una pregunta.
- Para apoderado: sugerir explicación doméstica y fuente oficial cuando corresponda.
- Para docente: indicar habilidad, actividad, evaluación y adaptación.
- No afirmar asesoría jurídica individual.
- No inventar leyes, artículos, plazos, sanciones ni competencias.
- En conflictos, priorizar seguridad, protección y resolución pacífica.
- En ciudadanía digital, no solicitar contraseñas, direcciones ni datos privados.

# Votación, mayoría y respeto a minorías

> **Estado de referencia:** colección organizada el 2026-07-14. Los capítulos jurídicos explican conceptos escolares generales. No sustituyen asesoría legal. Toda norma, autoridad, procedimiento, plazo o requisito vigente debe comprobarse en fuentes oficiales actualizadas, especialmente Biblioteca del Congreso Nacional, Poder Judicial, Congreso Nacional, municipalidades, Servel y organismos competentes.


## Respuesta breve

**Votación, mayoría y respeto a minorías** se estudia para comprender cómo las personas conviven, ejercen derechos, cumplen responsabilidades, participan en decisiones y resuelven conflictos mediante reglas e instituciones.

## Explicación clara

La educación ciudadana no consiste solo en memorizar nombres de autoridades. Ayuda a entender tres relaciones:

1. **Persona y comunidad:** cada persona tiene dignidad, derechos y responsabilidades.
2. **Reglas e instituciones:** las normas organizan la convivencia y las instituciones cumplen funciones que deben estar limitadas por el derecho.
3. **Participación y bien común:** una democracia necesita personas informadas, capaces de dialogar, organizarse, votar cuando corresponda y controlar pacíficamente a las autoridades.

En **Votación, mayoría y respeto a minorías**, se debe diferenciar entre una norma cotidiana, una regla escolar, una ley y una regla constitucional. También se debe distinguir entre lo que una institución **debería hacer según una explicación general** y lo que establece actualmente una norma específica.

## Conceptos clave

- **mayoría:** concepto necesario para comprender y buscar este tema.
- **minoría:** concepto necesario para comprender y buscar este tema.
- **voto:** concepto necesario para comprender y buscar este tema.
- **respeto:** concepto necesario para comprender y buscar este tema.

## Ejemplo cotidiano

Imagina que un curso necesita decidir cómo usar un espacio común. Primero escucha distintas propuestas. Luego acuerda criterios: seguridad, acceso para todos y cuidado de materiales. Puede votar, pero también debe escuchar a quienes prefieren otra opción.

La votación permite decidir. El respeto a las reglas y a las personas permite que la decisión sea democrática.

## Ejemplo adaptado para estudiantes con TEA

Se presenta una secuencia predecible:

```text
1. ¿Qué ocurrió?
2. ¿Qué regla o derecho se relaciona?
3. ¿Quiénes participan?
4. ¿Qué opciones existen?
5. ¿Qué solución evita daño y respeta derechos?
```

Ejemplo:

> “Dos estudiantes quieren usar el mismo material. No se decide quién es mejor. Se puede usar un turno, compartir, buscar otro material o pedir mediación.”

El estudiante puede responder eligiendo una opción, ordenando pasos o completando una tabla. No es obligatorio relatar conflictos personales.

## Diferencias importantes

| Concepto | Explicación escolar |
|---|---|
| Norma | Regla que orienta una conducta en un grupo o situación. |
| Ley | Norma jurídica aprobada mediante un procedimiento oficial y obligatoria dentro de su ámbito. |
| Derecho | Facultad o protección reconocida a las personas. |
| Deber o responsabilidad | Conducta esperada para respetar derechos, reglas y bien común. |
| Institución | Organización con funciones definidas. |
| Justicia | Búsqueda de decisiones imparciales, justificadas y respetuosas de derechos. |

## Actividad breve

### Caso para analizar

Una comunidad escolar quiere disminuir los mensajes ofensivos en un grupo digital.

1. Identifica el problema.
2. Reconoce derechos involucrados.
3. Propón una regla clara.
4. Indica qué debe ocurrir si alguien necesita ayuda.
5. Explica cómo evitar una sanción humillante o desproporcionada.

## Preguntas de práctica

### Inicial

1. ¿Cuál es la idea principal del tema?
2. Nombra un derecho o responsabilidad relacionado.
3. ¿Qué institución, persona o grupo podría intervenir?

### Intermedio

4. ¿Qué diferencia hay entre una opinión y una regla?
5. ¿Cómo se puede resolver el conflicto sin violencia?
6. ¿Qué información necesitarías verificar?

### Desafío

7. ¿La solución propuesta respeta a la mayoría y a la minoría?
8. ¿Qué control debería existir sobre quien toma la decisión?
9. ¿Cómo cambiaría el caso en internet?

## Respuestas esperadas

- Debe identificar derechos, responsabilidades, reglas o instituciones pertinentes.
- Una solución democrática debe poder explicarse, admitir revisión y evitar discriminación.
- Si se habla de leyes, autoridades, competencias o procedimientos vigentes, debe consultarse una fuente oficial.
- Resolver pacíficamente no significa aceptar abuso: puede ser necesario pedir protección o recurrir a una institución.

## Errores comunes

- Creer que democracia significa que la mayoría puede hacer cualquier cosa.
- Confundir derecho con deseo.
- Pensar que toda regla es una ley.
- Suponer que una autoridad no tiene límites.
- Confundir justicia con venganza.
- Compartir datos personales para “demostrar” un problema.
- Dar consejos legales específicos sin revisar la legislación vigente.
- Presentar una institución de manera simplificada como si resolviera todos los casos.

## Cómo explicarlo si el estudiante no entiende

> “Vivir con otras personas requiere reglas. Las reglas deben proteger a las personas. Cuando hay un problema, hablamos, revisamos derechos y buscamos una solución justa.”

## Para apoderados

Use situaciones cotidianas: turnos, permisos, mensajes digitales, cuidado de espacios y decisiones familiares. Evite explicar ciudadanía únicamente como obediencia. Pregunte también quién participa, qué derecho se protege y cómo se revisa una decisión injusta.

## Para docentes

Trabaje con casos breves, roles claros y fuentes institucionales. Antes de un debate, entregue:

- conceptos;
- reglas de diálogo;
- evidencia mínima;
- opciones de participación;
- forma de cerrar y evaluar.

No obligue a estudiantes a representar violencia, discriminación o victimización.

## Mini resumen

- Todas las personas tienen dignidad.
- Los derechos necesitan protección y responsabilidades.
- Las instituciones tienen funciones y límites.
- La participación requiere información, respeto y control.
- Los datos jurídicos actuales deben verificarse.

## Palabras clave para búsqueda local

mayoría, minoría, voto, respeto

## Temas relacionados

### Cursos anteriores

normas, respeto y convivencia

### Cursos posteriores

democracia, derechos, instituciones y participación

## Reglas para Chat Escolar

- Adaptar lenguaje y extensión al curso activo.
- Para estudiante: respuesta clara, ejemplo y una pregunta.
- Para apoderado: sugerir explicación doméstica y fuente oficial cuando corresponda.
- Para docente: indicar habilidad, actividad, evaluación y adaptación.
- No afirmar asesoría jurídica individual.
- No inventar leyes, artículos, plazos, sanciones ni competencias.
- En conflictos, priorizar seguridad, protección y resolución pacífica.
- En ciudadanía digital, no solicitar contraseñas, direcciones ni datos privados.

# Conflicto, negociación y mediación

> **Estado de referencia:** colección organizada el 2026-07-14. Los capítulos jurídicos explican conceptos escolares generales. No sustituyen asesoría legal. Toda norma, autoridad, procedimiento, plazo o requisito vigente debe comprobarse en fuentes oficiales actualizadas, especialmente Biblioteca del Congreso Nacional, Poder Judicial, Congreso Nacional, municipalidades, Servel y organismos competentes.


## Respuesta breve

**Conflicto, negociación y mediación** se estudia para comprender cómo las personas conviven, ejercen derechos, cumplen responsabilidades, participan en decisiones y resuelven conflictos mediante reglas e instituciones.

## Explicación clara

La educación ciudadana no consiste solo en memorizar nombres de autoridades. Ayuda a entender tres relaciones:

1. **Persona y comunidad:** cada persona tiene dignidad, derechos y responsabilidades.
2. **Reglas e instituciones:** las normas organizan la convivencia y las instituciones cumplen funciones que deben estar limitadas por el derecho.
3. **Participación y bien común:** una democracia necesita personas informadas, capaces de dialogar, organizarse, votar cuando corresponda y controlar pacíficamente a las autoridades.

En **Conflicto, negociación y mediación**, se debe diferenciar entre una norma cotidiana, una regla escolar, una ley y una regla constitucional. También se debe distinguir entre lo que una institución **debería hacer según una explicación general** y lo que establece actualmente una norma específica.

## Conceptos clave

- **negociación:** concepto necesario para comprender y buscar este tema.
- **mediación:** concepto necesario para comprender y buscar este tema.
- **interés:** concepto necesario para comprender y buscar este tema.
- **acuerdo:** concepto necesario para comprender y buscar este tema.

## Ejemplo cotidiano

Imagina que un curso necesita decidir cómo usar un espacio común. Primero escucha distintas propuestas. Luego acuerda criterios: seguridad, acceso para todos y cuidado de materiales. Puede votar, pero también debe escuchar a quienes prefieren otra opción.

La votación permite decidir. El respeto a las reglas y a las personas permite que la decisión sea democrática.

# Bienes públicos y patrimonio común

> **Estado de referencia:** colección organizada el 2026-07-14. Los capítulos jurídicos explican conceptos escolares generales. No sustituyen asesoría legal. Toda norma, autoridad, procedimiento, plazo o requisito vigente debe comprobarse en fuentes oficiales actualizadas, especialmente Biblioteca del Congreso Nacional, Poder Judicial, Congreso Nacional, municipalidades, Servel y organismos competentes.


## Respuesta breve

**Bienes públicos y patrimonio común** se estudia para comprender cómo las personas conviven, ejercen derechos, cumplen responsabilidades, participan en decisiones y resuelven conflictos mediante reglas e instituciones.

## Explicación clara

La educación ciudadana no consiste solo en memorizar nombres de autoridades. Ayuda a entender tres relaciones:

1. **Persona y comunidad:** cada persona tiene dignidad, derechos y responsabilidades.
2. **Reglas e instituciones:** las normas organizan la convivencia y las instituciones cumplen funciones que deben estar limitadas por el derecho.
3. **Participación y bien común:** una democracia necesita personas informadas, capaces de dialogar, organizarse, votar cuando corresponda y controlar pacíficamente a las autoridades.

En **Bienes públicos y patrimonio común**, se debe diferenciar entre una norma cotidiana, una regla escolar, una ley y una regla constitucional. También se debe distinguir entre lo que una institución **debería hacer según una explicación general** y lo que establece actualmente una norma específica.

## Conceptos clave

- **patrimonio:** concepto necesario para comprender y buscar este tema.
- **bien público:** concepto necesario para comprender y buscar este tema.
- **cuidado:** concepto necesario para comprender y buscar este tema.
- **comunidad:** concepto necesario para comprender y buscar este tema.

## Ejemplo cotidiano

Imagina que un curso necesita decidir cómo usar un espacio común. Primero escucha distintas propuestas. Luego acuerda criterios: seguridad, acceso para todos y cuidado de materiales. Puede votar, pero también debe escuchar a quienes prefieren otra opción.

La votación permite decidir. El respeto a las reglas y a las personas permite que la decisión sea democrática.

## Ejemplo adaptado para estudiantes con TEA

Se presenta una secuencia predecible:

```text
1. ¿Qué ocurrió?
2. ¿Qué regla o derecho se relaciona?
3. ¿Quiénes participan?
4. ¿Qué opciones existen?
5. ¿Qué solución evita daño y respeta derechos?
```

Ejemplo:

> “Dos estudiantes quieren usar el mismo material. No se decide quién es mejor. Se puede usar un turno, compartir, buscar otro material o pedir mediación.”

El estudiante puede responder eligiendo una opción, ordenando pasos o completando una tabla. No es obligatorio relatar conflictos personales.

## Diferencias importantes

| Concepto | Explicación escolar |
|---|---|
| Norma | Regla que orienta una conducta en un grupo o situación. |
| Ley | Norma jurídica aprobada mediante un procedimiento oficial y obligatoria dentro de su ámbito. |
| Derecho | Facultad o protección reconocida a las personas. |
| Deber o responsabilidad | Conducta esperada para respetar derechos, reglas y bien común. |
| Institución | Organización con funciones definidas. |
| Justicia | Búsqueda de decisiones imparciales, justificadas y respetuosas de derechos. |
