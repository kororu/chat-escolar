# Control de calidad

## Lista de comprobación

- [ ] El curso y la asignatura son correctos.
- [ ] La alineación está declarada.
- [ ] El texto es original.
- [ ] Se ubican tiempo y espacio.
- [ ] Hay causas, consecuencias, cambio y continuidad.
- [ ] Se reconocen actores diversos.
- [ ] Las fuentes se contextualizan.
- [ ] No existe lenguaje discriminatorio.
- [ ] Los temas sensibles siguen el protocolo.
- [ ] El código OA exacto fue revisado antes de publicación institucional.
- [ ] No se indexan compendios duplicados.
