# Enciclopedia de 6° básico — Chat Escolar

Base de conocimiento original, modular e inclusiva para una aplicación local con React, FastAPI y futura IA mediante Ollama.

## Contenido

- 97 capítulos curriculares principales.
- 25 capítulos de Lenguaje y Comunicación.
- 24 capítulos de Matemática.
- 22 capítulos de Ciencias Naturales.
- 26 capítulos de Historia, Geografía y Ciencias Sociales.
- Bancos de preguntas y glosarios.
- Evaluaciones diagnósticas con respuestas.
- Apoyos TEA y lectura fácil.
- Compendios para lectura humana.
- Catálogo y 1913 fragmentos JSONL para RAG.

## Instalación en el proyecto

Copiar esta carpeta en:

```text
chat-escolar/contenidos/sexto_basico/
```

Indexar principalmente:

- `lenguaje/`
- `matematica/`
- `ciencias_naturales/`
- `historia_geografia/`

Excluir `compendios/` para evitar duplicados.
