# Índice maestro — Enciclopedia de 6° básico para Chat Escolar

## Propósito

Esta base modular ofrece contenido curricular original para respuestas locales de Chat Escolar, con profundidad apropiada para estudiantes generalmente de 11 a 12 años, lectura fácil, apoyo TEA, ejemplos, actividades y respuestas explicadas.

## Uso recomendado

1. Indexar los capítulos modulares de las cuatro asignaturas.
2. Excluir `compendios/` de la búsqueda RAG para evitar duplicados.
3. Usar metadatos de curso, asignatura, eje, OA y tema como filtros.
4. Recuperar entre 3 y 6 fragmentos y generar una respuesta adaptada.
5. Registrar los identificadores de archivos usados.
