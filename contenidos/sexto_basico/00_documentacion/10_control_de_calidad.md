# Control de calidad

- [ ] Curso, asignatura, eje y tema correctos.
- [ ] OA verificado o marcado para revisión.
- [ ] Contenido original y sin reproducción de textos escolares.
- [ ] Explicación central comprensible para sexto básico.
- [ ] Ejemplo correcto y actividad realizable.
- [ ] Preguntas y respuestas revisadas.
- [ ] Unidades, cálculos, fechas y conceptos comprobados.
- [ ] Adaptación TEA sin infantilización.
- [ ] Tema sensible tratado con privacidad y seguridad.
- [ ] Enlaces internos válidos.
- [ ] Metadatos listos para indexación.
- [ ] Fecha y versión registradas.
