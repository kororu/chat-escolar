# Tema: Generalizaciones con letras y ecuaciones

## Respuesta breve

Las letras representan números variables o desconocidos y permiten escribir relaciones generales.

## Qué aprenderás

- Comprender y explicar generalizaciones con letras y ecuaciones.
- Aplicar lo aprendido en un ejemplo apropiado para 6° básico.
- Fundamentar una respuesta usando vocabulario y evidencia del tema.

## Objetivos de Aprendizaje relacionados

- MA06 OA 10

## Ideas fundamentales

- Una letra puede tomar distintos valores.
- La expresión no siempre es una ecuación.
- La generalización describe muchos casos.
- Las operaciones mantienen su significado.

## Conceptos importantes

### Expresión algebraica

combinación de números, letras y operaciones.

### Variable

letra que representa valores que cambian.

### Incógnita

valor desconocido que se busca.

### Generalización

regla válida para un conjunto de casos.

## Explicación completa para estudiante

La frase “el doble de un número más tres” se escribe 2n+3. No es lo mismo que 2(n+3). Una ecuación incluye igualdad, por ejemplo 2n+3=15. Las letras ayudan a expresar perímetros, costos o patrones. Se debe definir qué representa cada una y conservar unidades.

## Ejemplo explicado

El perímetro de un cuadrado de lado l es P=4l. La fórmula funciona para cualquier longitud válida.

## Preguntas de práctica

1. ¿Qué representa n en 5n?
2. ¿Es 3x+2 una ecuación?
3. ¿Qué diferencia 2n+3 de 2(n+3)?
4. ¿Para qué generalizar?

## Respuestas esperadas y explicación

1. **Respuesta:** Un número variable multiplicado por 5.
2. **Respuesta:** No, es una expresión sin igualdad.
3. **Respuesta:** En la segunda se duplica la suma completa.
4. **Respuesta:** Para describir muchos casos con una regla.

## Errores comunes

- Interpretar letras como objetos.
- Omitir paréntesis.
- Usar una letra sin definir.
