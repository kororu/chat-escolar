# Changelog del proyecto

## Desarrollo 0.1.x

- Se creó la base local React/Vite + FastAPI + SQLite.
- Se incorporaron perfiles, historial, favoritos, pendientes y avatares locales.
- Se añadieron contenidos curriculares de 1.º a 8.º básico, búsqueda con puntuación, normalización y detección de materia.
- Se mejoraron respuestas pedagógicas, adaptación por curso, lectura fácil y contexto conversacional.
- Se integró Nexo, variantes por materia, globos, demo y preguntas sugeridas compactas.
- Se agregó Ollama opcional, configuración local, procedencia de respuestas y fallback básico.
- Se añadieron scripts de Windows, verificación de entorno y optimización del índice local.
- Se consolidó documentación para usuarios, desarrollo, UX y preparación de 1.0.

## Pendiente para 1.0

Pack portable, instalador Windows, pruebas finales, licencia y release `v1.0.0`.
