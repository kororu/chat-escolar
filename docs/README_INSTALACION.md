# Enciclopedias de 2°, 3°, 4°, 7° y 8° básico — Chat Escolar

Paquete para completar la cobertura de Educación Básica de 1° a 8° junto con las enciclopedias ya creadas de 1°, 5° y 6° básico.

## Instalación directa

Este ZIP contiene una carpeta `contenidos`. Extraiga el paquete en la raíz del proyecto:

```text
C:\Users\Ariel\Documents\proyectos\chat-escolar
```

Windows debería combinar la carpeta nueva con `contenidos` sin reemplazar los cursos existentes.

Resultado esperado:

```text
chat-escolar/
└── contenidos/
    ├── primero_basico/
    ├── segundo_basico/
    ├── tercero_basico/
    ├── cuarto_basico/
    ├── quinto_basico/
    ├── sexto_basico/
    ├── septimo_basico/
    └── octavo_basico/
```

## Indexación

Indexar inicialmente solo las carpetas de asignaturas de cada curso. Excluir `compendios`, `evaluaciones`, `bancos`, `apoyo_inclusivo` y `00_documentacion` hasta configurar colecciones separadas.
